# راهنمای راه‌اندازی وبهوک - افزونه ربات فروشنده

## پیش‌نیازها

قبل از شروع، مطمئن شوید:

- **SSL فعال است** — سایت شما حتماً باید HTTPS داشته باشد. بدون SSL وبهوک کار نمی‌کند (تلگرام و بله فقط HTTPS قبول می‌کنند).
- **پایوند یکتای وردپرس** روی حالت "نام نوشته" (Post name) یا هر چیزی غیر از "ساده" (Plain) باشد:
  - **تنظیمات → پایوندهای یکتا → نام نوشته**
  - اگر روی "ساده" باشد REST API کار نمی‌کند و وبهوک فعال نمی‌شود.

---

## مرحله ۱: ساخت ربات

### تلگرام:
1. در تلگرام `@BotFather` را سرچ کنید
2. `/newbot` بزنید
3. یک **نام** برای ربات بدید (مثلاً: فروشگاه من)
4. یک **یوزرنیم** بدید (مثلاً: `myshop_bot`) — باید به `bot` ختم شود
5. BotFather یک **توکن** بهتون میده مثل: `7123456789:AAHxxx...`
6. توکن رو کپی کنید

### بله (Bale):
1. در بله `@BotFather` را سرچ کنید
2. همان مراحل بالا
3. توکن رو کپی کنید

---

## مرحله ۲: تنظیمات عمومی

1. در پیشخوان وردپرس برید به **ربات فروشنده → تنظیمات**
2. تب **عمومی** را باز کنید
3. **پلتفرم** را انتخاب کنید: `تلگرام` یا `بله`
4. **نام فروشگاه** را تکمیل کنید
5. ذخیره کنید

---

## مرحله ۳: تنظیمات ربات

1. تب **ربات** را باز کنید
2. **توکن ربات** — توکنی که از BotFather گرفتید را وارد کنید
3. **ابتدا ذخیره کنید** (دکمه ذخیره تنظیمات)
4. **تست اتصال** — دکمه «تست اتصال» را بزنید
   - اگر ✅ نشان داد: توکن درسته
   - اگر ❌ نشان داد: توکن اشتباهه یا پلتفرم اشتباه انتخاب شده
5. **یوزرنیم ربات** — بدون `@` وارد کنید (مثلاً: `myshop_bot`)
6. ذخیره کنید

---

## مرحله ۴: فعال‌سازی وبهوک (مهم‌ترین بخش)

> **وبهوک** یعنی تلگرام/بله هر پیامی که کاربر به ربات بفرسته، فوری به سایت شما POST می‌کنه.

### فعال‌سازی:
1. در همان تب **ربات**، بخش **وبهوک** را پیدا کنید
2. دکمه **«فعال‌سازی وبهوک»** را بزنید
3. اگر ✅ **فعال شد** نشان داد: تبریک! وبهوک فعال شده
4. صفحه ریلود می‌شود و وضعیت وبهوک **✅ فعال** نشان می‌دهد

### اگر خطا گرفتید:

#### ❌ خطا: SSL Certificate
- گواهی SSL سایت معتبر نیست. باید از Let's Encrypt یا SSL معتبر استفاده کنید
- Self-signed SSL قبول نیست

#### ❌ خطا: Connection timeout / Bad webhook
- سایت شما از اینترنت قابل دسترسی نیست
- اگر سایت در localhost یا شبکه داخلی است، وبهوک کار نمی‌کند
- باید سایت روی هاست واقعی باشد

#### ❌ خطا: 404
- پایوند یکتا روی "ساده" است → به "نام نوشته" تغییر بدید
- یکبار پایوند یکتا رو ذخیره کنید (بدون تغییر هم ذخیره کنید تا rewrite rules فلاش بشه)

#### ❌ خطا: 401 / Unauthorized
- توکن ربات اشتباه وارد شده
- اول توکن رو ذخیره کنید، بعد وبهوک بزنید

#### ❌ خطا: Wrong response from webhook
- پلاگین‌های امنیتی (مثل Wordfence, iThemes) ممکنه REST API رو بلاک کنن
- این URL رو در مرورگر تست کنید: `https://your-site.com/wp-json/robot-forooshande/v1/`
- اگر 404 یا خطا خورد، REST API مشکل داره

### تست دستی وبهوک:
اگر می‌خواید مطمئن بشید WordPress REST API کار می‌کنه، این آدرس رو در مرورگر باز کنید:
```
https://your-site.com/wp-json/
```
اگر یک JSON طولانی نشان داد، REST API فعاله. اگر ۴۰۴ خورد، مشکل از پایوند یکتا است.

## مرحله ۵: چت آیدی ادمین

1. ربات `@userinfobot` را در تلگرام استارت کنید (یا ربات مشابه در بله)
2. چت آیدی عددی خودتون رو بگیرید
3. در تب **ربات → چت آیدی ادمین‌ها** وارد کنید
4. اگر چند ادمین دارید، با کاما جدا کنید: `123456,789012`
5. ذخیره کنید

> ادمین‌ها به دکمه 🔧 پنل مدیریت در ربات دسترسی دارند و اعلان‌های سفارش جدید، پرداخت و ... دریافت می‌کنند.

---

## مرحله ۶: تست نهایی

1. در تلگرام/بله ربات خودتون رو سرچ کنید
2. `/start` بزنید
3. باید پیام خوش‌آمدگویی و درخواست شماره تلفن بیاد
4. شماره تلفن رو بفرستید
5. منوی اصلی با دکمه‌های فروشگاه، سبد خرید و ... باید نمایش داده بشه

اگر `/start` زدید و هیچ چیزی نیامد:
- وبهوک فعال نشده
- توکن اشتباهه
- SSL مشکل داره

---

## مشکلات رایج و راه‌حل

| مشکل | راه‌حل |
|---|---|
| ربات جواب نمیده | وبهوک فعال نیست — دوباره فعال‌سازی کنید |
| وبهوک فعال نمیشه | SSL بررسی کنید + پایوند یکتا "نام نوشته" باشد |
| تست اتصال خطا میده | توکن رو دوباره از BotFather بگیرید |
| دکمه‌ها کار نمیکنن | مطمئن بشید وبهوک فعاله و SSL معتبره |
| REST API 404 میده | تنظیمات → پایوند یکتا → ذخیره (بدون تغییر) |
| پلاگین امنیتی بلاک میکنه | REST API رو در تنظیمات پلاگین امنیتی آزاد کنید |
| سایت روی localhost | وبهوک فقط با آدرس عمومی کار میکنه — از ngrok استفاده کنید |

## ساختار URL وبهوک

URL وبهوک به این شکل ساخته می‌شود:
```
https://your-site.com/wp-json/robot-forooshande/v1/webhook/{hash}
```

- `{hash}` یک مقدار MD5 است که از توکن ربات + salt سایت ساخته میشه
- این hash امنیتی هست تا کسی نتونه درخواست جعلی بفرسته
- هر بار که وبهوک رو فعال کنید، hash جدید ساخته میشه

### فرایند فعال‌سازی وبهوک:
```
۱. کلیک روی «فعال‌سازی وبهوک»
   ↓
۲. افزونه hash = md5(توکن + wp_salt) می‌سازه
   ↓
۳. URL = /wp-json/robot-forooshande/v1/webhook/{hash}
   ↓
۴. درخواست setWebhook به API تلگرام/بله ارسال میشه
   ↓
۵. تلگرام/بله URL رو ذخیره میکنه
   ↓
۶. از این به بعد هر پیام کاربر → POST به این URL
   ↓
۷. افزونه پیام رو دریافت → پردازش → پاسخ
```

## غیرفعال‌سازی وبهوک

اگر نیاز بود وبهوک رو غیرفعال کنید:
1. دکمه **«غیرفعال‌سازی»** را بزنید
2. وضعیت به **❌ غیرفعال** تغییر می‌کنه
3. ربات دیگه پیام‌ها رو دریافت نمی‌کنه

> برای تغییر دامنه سایت، اول وبهوک رو غیرفعال کنید، دامنه رو عوض کنید، بعد دوباره فعال کنید.
